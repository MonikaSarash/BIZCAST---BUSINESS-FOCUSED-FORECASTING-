## 📌 Overview

BIZCAST is an AI-powered business forecasting system developed to predict future sales trends, revenue growth, and demand patterns using historical business data. The project uses advanced Time Series Forecasting with the Prophet model to help businesses make smart and data-driven decisions. It improves planning, inventory control, and resource management through accurate forecasting. 

---

## 🚀 Features

* 📈 Future Sales Prediction
* 💰 Revenue Forecasting
* 📊 Trend & Seasonality Analysis
* 📂 CSV / Excel Dataset Upload
* 📉 Interactive Graph Visualization
* 📑 Forecast Report Generation
* 🎯 MAE & RMSE Accuracy Evaluation
* 🌐 User-Friendly Web Dashboard

## 🧠 Machine Learning Model

The forecasting engine is built using the **Prophet Model**, which automatically detects:

* Trends
* Seasonality
* Holidays / Sudden Changes
* Future Forecast Patterns
